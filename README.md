# Hugo AWS Project

Website được xây dựng bằng Hugo static site generator.

## Cài đặt

1. Cài đặt Hugo: https://gohugo.io/installation/
2. Chạy server local: `hugo server`
3. Build site: `hugo`

## Cấu trúc

- `content/`: Nội dung markdown
- `static/`: Các file tĩnh (ảnh, CSS, JS)
- `themes/`: Hugo theme
- `config.toml`: Cấu hình Hugo
